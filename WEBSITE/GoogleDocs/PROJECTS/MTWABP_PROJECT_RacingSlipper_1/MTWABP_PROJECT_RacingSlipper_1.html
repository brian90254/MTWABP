Racing Slipper

DESIGN AND BUILD A CUSTOM SHOE

 

OVERVIEW:

This project is a step-by-step walk through of designing and building a custom shoe. These techniques can apply to many different designs, the one we’re demonstrating here we’re calling a “Racing Slipper”. The patterning process utilizes a shoe “last” which we apply masking tape to, draw directly onto that surface, then cut and flatten manually. So, no computer is required for this method, although we will demonstrate using open-source vector programs to assist in the design as an option.

  •  Minimal number of pattern pieces to keep the construction simple.
  •  Slip-on design, so no laces.
  •  AF1 soles should be easy to find in the custom shoe building community.
  •  How-to videos for each step.
