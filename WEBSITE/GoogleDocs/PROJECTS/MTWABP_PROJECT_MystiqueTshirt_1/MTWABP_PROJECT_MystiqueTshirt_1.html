Mystique T-Shirt

  

OVERVIEW:

This is a contemporary t-shirt that repurposes extra ink transfers from the Tattoo Mystique pinball game, providing additional opportunities to display and enjoy this artwork.

  •  The artwork for this custom pinball game was inspired by traditional vintage tattoo flash designs.
  •  An ink transfer process was utilized to apply each vignette individually, as if the wooden playfield itself was “tattooed”.
  •  Because of the experimental nature of this process, there were many unused transfer films left over, which were then repurposed into custom t-shirt designs.
  •  Those new composited designs are made available for personnel use on our GitHub page (links below).
