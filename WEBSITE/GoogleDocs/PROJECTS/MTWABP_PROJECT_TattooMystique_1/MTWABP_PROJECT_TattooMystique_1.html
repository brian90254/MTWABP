Tattoo Mystique

   

OVERVIEW:

“Tattoo Mystique” is a homebrew pinball machine featuring custom playfield layout, original artwork, and a novel set of game rules. The first version of this game debuted at the Seattle NWPAS in 2015, and a second reboot version was built nine years later and similarly debuted at NWPAS in 2024.

  •  Custom playfield layout, designed to be reminiscent of classic single-level games from the late 80’s.
  •  All original artwork on the playfield, plastics and backglass. The concept is to give the feel that the wooden playfield was “tattooed”.
  •  Novel ruleset, scoring, and mechanical features not seen in other games.
  •  Utilizes FAST Pinball hardware, and the open source Mission Pinball Framework (MPF) for creating game code.
